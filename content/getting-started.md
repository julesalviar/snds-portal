# Getting started

Welcome to the Info Guide — a step‑by‑step tutorial that explains every feature you’ll use to set up and manage your account. This page walks you through the complete process, from uploading files to configuring pre‑installed data, and explains what information is required and where to enter it. Follow the sections below to quickly get your account ready and ensure consistency.

## What this page covers
(*this guide is applicable only for school admin users*)
1. Uploading files: how to add profile documents, logos, and other required files.
2. Using Pre‑installed "Stakeholder" list: Use the built‑in stakeholder list for uniformity and consistency.
3. Unit Guide list (the master list of units).
4. Filling forms and inputting data: what fields are required, recommended formats, and where to enter additional information.
