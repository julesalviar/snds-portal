# Getting started

This is a sample docs page in plain Markdown.

## First steps

1. Open the [Docs](/docs) overview for the full index.
2. Use the search bar (or press `Ctrl+K` / `⌘K`) to find topics.
3. Use the sidebar to move between pages.

## Links and formatting

- **Bold** and *italic* work as usual.
- [Links](/docs) to other pages or the portal.
- Code: use `backticks` for inline code.
